Values and Data Types

Programming Language consist of keywords (function, if, return ..), values (13, 'abc', true), symbols (;, }, {, [ …) and operators (+, i,..) arranged in way (syntax) that make sense to computer. Values are "built-in objects" that computer evaluates to themselves and can be different types.
We can use special operator typeof to determine type of value. 

typeof 3

Numbers

123 // Integer
3.14159265358979 // Float number
-24.01 // Negative number
Infinity

Operations 

+, -, *, /,

2 + 2

Use parenthesis to specify operator precedence. 

(1 + 2) * (3 - 4) / 5

Text

Strings are characters enclosed inside quotes (single quotes ' , double quotes " or backticks `) 

'Sample single line text using quotes'

To escape quote character inside of the string we use \ in front of the quote character. 

`Sample multi line
  text using backticks \` `

We can use ${} when using backticks ` to insert another value inside string. 

`A bout de souffle, ${1960}`

We can also use + operator to join strings together. 

'Spaghetti' + ' with ' + 'tomato sauce'

Boolean

true

false

Mostly used in conditionals to evaluate truthiness of given expression. 

1 == 1

2 == 3

Objects

Key value pairs. 

{
  key: 'value',
  'also key' => 'another value',
  third_key: 12345
}

Collections

Array is defined using square brackets [] 

[33, 'A', 'Pizza', false, 2.45]

Array can also contain nested array 

[[1, 2], [3, 4], [[5, 6], 'b']]

Variables

Allow us to give values names. 

let a = 123;
let b = 'abc';

const keyword defines constant (Something that can not be changed later.) 

const C = 'Fish';

let a = 3;
let b = 7;

a + b;

Conditionals

if (condition) {
  /* code to run if condition is true */
} else {
  /* run some other code instead */
}

A condition represents something that is evaluated to true or false 

if (2 == 7) {
  'this will not be evaluated and result is undefined';
}

if ('a string' == 'another string') {
  'this will not be evaluated';
} else {
  'inside of the else branch will be evaluated';
}

let a = 1;
let b = 2;
let c = 3;

if (a == 3) {
  'a is equal to 3';
} else if (b == 3) {
  'b is equal to 3';
} else if (c == 3) {
  'c is equal to 3';
} else {
  'nothing equals to 3'
}

Conditions can be combined together using "and" operator && or "or" operator ||. 

2 == 2 && 3 == 3 || false;

https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/conditionals

Functions

Define piece of code that can be evaluated later. 

1: // function definition:
2: function meaningOfLife() {
3:   return 42;
4: }
5: 
6: // function execution
7: meaningOfLife();
  • Line 1: Comment ignored by computer
  • Line 2: Contains function name (meaningOfLife), empty list of parameters () and opening curly bracket of body of the function.
  • Line 3: Contains keyword return followed by value returned by the function when it gets executed.
  • Line 4: Closing bracket of function body and also end of the function definition
  • Line 7: Will cause the function being executed and return a value

Functions can have parameters 

function plusTwo(n) {
  return n + 2;
}

plusTwo(31);

We need to make sure that value we pass as argument make sense otherwise the result of evaluation can be unexpected. 

function plusFour(n) {
  return n + 4;
}

plusFour('seven');

For now we can consider two types of functions:

  • built-in functions
  • ours

Combination of these two allow us write useful programs.

https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/Functions
