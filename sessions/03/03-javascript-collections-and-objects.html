Objects

Definition: An object is a thing with properties and behaviour. In JavaScript objects are represented by curly brackets {} Which may contain properties. 

// object describing a chair
// assigned to local variable named "chair"
let chair = {
  colour: 'red',
  legs: 4,
  material: 'wood'
}

// object describing a shift with
// nested object describing an employer
let shift = {
  id: 21,
  start_time: '2022-09-27T09:00:00.000Z',
  end_time: '2022-09-27T17:00:00.000Z',
  employer: { id: 1, name: 'Acme Co.' }
}

// object describing a person
// including also behaviour "greet"
let person = {
  first_name: 'Jules',
  last_name: 'Verne',
  age: 27,
  greet: function () { return 'bonjour' }
}

// assign value to property of object
// shift.worker = person;
// shift.worker.greet();

We can represent different types (classes) of objects in JavaScript but for our purposes we are not going to focus on that. Classes are represented in language using words with capital first letter. (Example: 'abc' is an object of class String)

Needless to say that all values in JavaScript are objects and such have properties and also behaviour. 1

Property value are defined in terms of primitive values (numbers, strings, true, false, array..) and are accessed using dot and name of the property. person.age 2

Behaviour of object is defined using functions. Function defined on object is called method.

Example:

We can find out number of characters in a string by accessing his property length 

'Around the World in Eighty Days'.length;

We can convert string from lower case letter to upper case letters by invoking behavior using method .toUpperCase(). 

'Captain Nemo'.toUpperCase();

To find out what behaviour we can invoke on built-in objects is best to consult documentation. Example: String documentation on MDN website

Collections

Array: Sequence of values inside of square brackets [], seperated by commas. ([2, 3, 4]) Array's are also objects and have properties and behaviour. We can access nth element of array using square brackets and index. Item's in array are indexed from 0. 

let arry = [2, 3, 4, 5];

// arry.length;
// arry[0]; // => 2
// arry[2]; // => 4

We can manipulate the collection by invoking methods defined on Array object. 

let arry1 = [4, 2];
let arr2 = [8, 6];

// join two arrays together
// arry1.concat(arr2);

// sort the result
// arry1.concat(arr2).sort();

// reverse in oposite order
// arry1.concat(arr2).sort().reverse();

// check if array contains specific value
// arry1.includes(6);
// arry1.concat(arr2).includes(6);

More: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array#instance_methods

We can transform the collections by providing functions to methods as an argument.

Reduce (transform)

Create "single" value from collection. 

let ages = [17, 21, 33, 43];

ages.reduce(
  function (accumulated, number) {
    return accumulated + number;
  }
)

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/reduce
