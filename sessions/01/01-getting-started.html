Lunch And Learn Code Q4 2022

Agenda

  • Walktrough over the overall course agenda.
  • Define basic rules and format of the sessions.
  • Get your hands "dirty" with few examples and exercises. :)

Format of the sessions

  • 10-15 minutes presentation by lecturer (recorded)
  • 25-30 minutes individual work on exercises (split to small groups with assistance by available mentors)
  • 5-10 minutes (Demos, wrap up and Q&A)

You can attend one or both weekly session to work on the exercises and get help by mentors.

Prerequisites

  • Web Browser Firefox or Google Chrome
  • Zoom account

Outcome of the course

You will:

  • gain ability to understand code, concepts and development process
  • be able to reuse knowledge and code from this course to build your own tools

Reasons to learn coding:

  • make life easier
  • build apps and tools for fun and profit
  • enhance career

Programming

Is constant loop of Write code (programmer) -> Evaluate code (computer) -> Feedback (computer & programmer)
